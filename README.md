# ChickPal

An intelligent IoT-powered poultry management platform for the Linda Chick Automated Poultry Coop System.

## Features

- Live telemetry
- MQTT communication
- ESP32 integration
- Automated feeding
- Environmental monitoring
- Remote control
- Analytics
- Notifications
- Authentication
- AI-powered insights

## Technologies

- React
- Next.js
- TypeScript
- MQTT
- Firebase / Supabase
- TailwindCSS
